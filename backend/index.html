Meeting Assistant API

Welcome to the Meeting Assistant API. This API allows you to upload meeting recordings, get transcriptions, and extract tasks.

Available Endpoints:

Upload Meeting

Endpoint: POST /api/meetings/upload

Description: Upload an audio file to transcribe and extract tasks.

Request: Form data with 'meeting' file field

Response:

{
  "message": "File uploaded, transcribed, and tasks extracted successfully",
  "file": "filename",
  "path": "/path/to/file",
  "transcript": "Transcribed and corrected text",
  "rawTranscript": "Raw transcription from Whisper",
  "tasks": [
    {
      "name": "employee_name",
      "task": "task_description",
      "due": "YYYY-MM-DD"
    }
  ],
  "transcriptFile": "/path/to/transcript.txt"
}
